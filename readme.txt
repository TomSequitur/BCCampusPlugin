Provides Textbook custom post type and Institutions taxonomy.  Also provides four Gutenberg blocks: author, license date, publication date, and feature display.

== Installation ==

1. Upload `bccampus-textbooks/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create Textbook posts and attached the provided Gutenberg blocks.
